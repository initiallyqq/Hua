便签页面概念图

Windows 11 风格桌面应用便签页面，现代极简主义办公美学，Material Design 3 与 Apple Human Interface 深度融合设计。界面尺寸约 1440×900px，整体采用与首页完全统一的视觉体系。背景颜色 #F8FAFC，主内容区域采用 #FFFFFF 纯白色卡片设计，搭配极其微弱的大面积弥散阴影与 24~28px 超大圆角。字体采用 Inter / SF Pro Display，整体视觉参考 Vercel Dashboard、Linear、Raycast、Apple Notes 与 Notion Calendar，通过克制的色彩、大量留白和清晰的信息层级塑造高级 AI 办公软件质感。

界面采用左侧便签管理区、中部内容编辑区与右侧实时预览区三栏布局结构。除最左全局图标侧边栏外，三栏宽度按 1 : 3.2 : 3.2 分配，使便签管理栏保持轻量，中间 Markdown 源码区与右侧渲染预览区保持等宽。整体设计理念强调「沉浸式创作」与「即时反馈」，弱化工具存在感，将内容本身作为整个页面的核心视觉焦点。三栏区域通过统一留白与一致的视觉语言形成协调关系，编辑区与预览区保持相同视觉权重，共同构成完整的 AI 写作工作空间。

左侧区域由最左全局图标侧边栏与便签管理栏组成。最左全局侧边栏颜色以首页为准，使用 #FCFCFD 的极简浅色底，不显示右边框；便签管理栏使用页面原有浅灰背景 #F8FAFC，并通过极细分割线与编辑区建立层级。顶部显示当前笔记本名称「日报」，右侧保留极简三点菜单入口。下方为搜索区域，采用圆角输入框设计，背景颜色略深于页面背景，通过极细边界与留白建立层级关系。整体视觉克制而安静，不出现明显描边与复杂装饰。

搜索区域下方为便签列表。列表采用纵向卡片式排列方式，每个便签展示标题、更新时间与内容摘要。当前选中内容采用极浅蓝色背景进行状态强调，同时通过字重变化增强识别度；未选中内容保持纯白背景与低对比度展示。列表风格参考 Apple Notes 与 Linear 的结合体，不强调任务管理属性，而强调知识记录与长期积累。整体区域如同一面数字便签墙，但经过高度秩序化整理，呈现安静、理性且具有高级感的信息管理体验。底部保留大量空白区域，使界面始终保持轻盈呼吸感。

中央区域为核心创作空间，是整个页面唯一的视觉中心。整体采用与首页收益卡片相同的设计语言，背景保持纯白色画布效果，通过微弱阴影与大量留白形成悬浮于桌面的空气感。编辑区左侧直接衔接便签管理栏，左上角与左下角不使用圆角，避免相邻栏位之间出现多余切角。页面不模拟真实纸张，不使用米白色阅读背景，不强调传统文档质感，而是强调 AI 原生工作空间的现代数字化表达。

顶部保留极简工具栏设计，仅包含少量高频操作入口。所有图标采用统一线性风格，间距宽松，视觉重量极低。工具栏更像内容创作过程中的辅助层，而非传统办公软件的功能中心。

内容区域明确为 Markdown 源码编辑区，而不是富文本渲染区。中间栏直接展示原始 Markdown 文本，包括一级标题、段落、引用符号、代码围栏与表格语法；整体采用等宽字体、舒适行高与纯白画布，使用户能够清晰看到内容结构本身。编辑器支持 Markdown 输入与 AI 智能补全，但补全不改变源码编辑的本质，视觉体验更接近 Cursor、Obsidian Source Mode 与现代 AI 原生 Markdown 编辑器。整个编辑区域不存在明显边框与功能分区，源码如同自然流动于页面之中，营造专注且连续的写作状态。

代码内容采用独立信息卡片呈现：

* 背景颜色 #F5F7FA
* 20px 大圆角
* 柔和阴影
* 等宽字体排版
* 极轻量语法高亮

引用内容通过左侧浅灰色竖线建立层级关系：

* 边框颜色 #DCE3EB
* 保持纯净背景
* 不使用强调色块

图片内容自动适配页面宽度：

* 统一圆角处理
* 保持舒适留白
* 阴影极其克制

表格内容遵循 Apple 风格极简规范：

* 使用细线分隔
* 不使用复杂边框
* 保持统一间距与阅读节奏

整体体验更接近现代知识库产品，而非传统 Markdown 编辑器或开发工具界面。

右侧区域为 Markdown 渲染预览区，与中央源码编辑区保持完全统一的视觉体系。整体采用独立阅读画布设计，不使用明显边框与分割线，而是通过空间关系建立层级。背景保持纯白色，与首页统计卡片属于同一设计语言。

Markdown 内容经过精致排版渲染：

* 一级标题采用 32~36px 字号
* 二级标题采用 24~28px 字号
* 三级标题采用 20~22px 字号
* 标题字重明显高于正文
* 正文保持 16px 字号
* 行高约为 1.8 倍
* 最大阅读宽度控制在 720px 左右

整体阅读体验接近 Apple Notes 阅读模式、Craft 阅读视图与 Notion Preview 的融合体。

代码块、引用内容、图片与表格均遵循统一视觉规范：

* 间距一致
* 对齐精准
* 无视觉噪声
* 保持内容连续阅读体验

预览区域并非传统 Markdown Preview，而更接近最终内容呈现画布，使创作与阅读形成即时反馈闭环。

页面强调「沉浸式写作」与「即时阅读反馈」两种核心体验。编辑区与预览区采用实时同步机制：

* 编辑时预览同步更新
* 编辑滚动与预览滚动实时对应
* 动画过渡时间约 150~200ms
* 使用 Apple 风格缓动曲线

所有悬浮状态均极其克制：

* Hover 仅出现轻微背景变化
* 不使用明显描边
* 不出现夸张动画效果

界面中的所有圆角、阴影与间距保持统一设计语言：

* 主容器圆角 28px
* 次级容器圆角 20px
* 输入框圆角 16px
* 图标按钮圆角 14px

阴影采用超低透明度大面积扩散阴影：

* 无明显方向性
* 不产生厚重卡片感
* 营造悬浮于桌面的空气感

色彩系统与首页完全统一。背景采用 #F8FAFC，主内容区域采用 #FFFFFF，文本颜色使用 Slate 灰阶体系。交互高亮统一采用浅蓝色系，选中状态使用极低饱和度蓝色背景；AI 功能与智能建议采用浅绿色作为辅助强调；其余颜色全部后退处理。整个页面几乎不存在高饱和颜色与强烈视觉冲突，通过蓝、灰、绿三色体系建立统一产品语言，使首页、便签页、回忆本与桌面组件形成完整且连续的视觉品牌体验。

整体界面不存在任何多余视觉干扰。内容始终作为唯一焦点，其余元素全部主动后退。通过超大留白、极细线条、柔和阴影与克制交互营造安静、专注、高级的 AI 原生创作体验，呈现出 Apple Dashboard、Vercel Analytics、Linear 与现代知识库产品融合后的下一代桌面生产力工具风格。

视觉关键词：Minimal、Calm、Premium、Productivity、Focus、Silent Luxury、AI Workspace、Apple Dashboard、Linear Design、Vercel Analytics、Knowledge Management、Real-time Preview、Modern Writing Experience。
