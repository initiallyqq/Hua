# Floral Notepaper

A lightweight, elegant, and modern desktop sticky note app built with Tauri 2 + React.

## Features

- Markdown editing and preview
- Quick note window
- Pin mode
- Import and export `.md` files

## Use Cases

- Quickly capture ideas and temporary notes
- Use it as a desktop sticky note or to-do list
- Copy, organize, and review notes anytime

## Download

Download the latest version from [GitHub Releases](https://github.com/Achilng/floral-notepaper/releases).

## Building from Source

### Prerequisites

- [Node.js](https://nodejs.org/) 18+
- [Rust](https://www.rust-lang.org/tools/install)
- [Tauri CLI 2](https://tauri.app/)

### Steps

```bash
git clone https://github.com/Achilng/floral-notepaper.git
cd floral-notepaper
npm install
npm run tauri dev
```

Production build:

```bash
npm run tauri build
```
